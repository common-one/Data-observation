### 二 安装
新建一个空的文件夹作为项目目录，并在目录下执行
```bash
yarn create umi
```
or
```bash
npm create umi
```
选择 ant-design-pro：
Ant Design Pro 脚手架将会自动安装。

### 三 本地开发
安装依赖
```bash
npm install
```
启动
```bash
npm start
```
